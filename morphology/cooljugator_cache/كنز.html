Sheriff said that it's a real treasure trove down there so I've got dibs on the old body spray, and 'Jugs' magazines

! ألا يريد أيّ أحد أن يرى إذا كان هناك فعلاً كنز

Doesn't anyone want to see if there actually is a treasure?

! أي كنز ؟

If there ain't no treasure, don't pay.
