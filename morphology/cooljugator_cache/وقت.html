Examples of وقت

حدّدوا وقت؟

They set a time?

حسنا,لم الإتفاق على وقت ان كنت لن تلتزم به؟

Well, why set a time if you're not gonna stick to it?

لذلك عليك انت تعلم انه عندما احدد وقت هنالك سبب

So you should know that when I set a time line, there's a reason.

هل قمت بتحديد وقت معين

Did you set a timer?

...عندما كنا صغار ...وكنا سنخرج برحلة بالسيارة ...وهو يحدد وقتاً

When we were kids and we were going on a car trip, he would set a time--

السيدة تايلور كنت اتسائل اذا كان بإمكاننا ان نحجز وقتاً لمقابلتك

Mrs. Taylor, I was wondering if we could set a time for your Torch interview.

عندما أحدد وقتاً تلتزمين به

When I set a time, you stick to it.
